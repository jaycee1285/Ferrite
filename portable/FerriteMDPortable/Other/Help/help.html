Ferrite MD Portable

Ferrite is a fast, lightweight text editor for Markdown, JSON, CSV, and more. Built in Rust with egui, it offers live preview, syntax highlighting, mermaid diagram rendering, and an integrated terminal.

Portable Mode: This portable version stores all settings and data in the Data folder within the application directory. No files are written to the host system. You can carry it on a USB drive and use it on any Windows PC.

Getting Started

  • Launch FerriteMDPortable.exe from the main application folder.
  • Open any .md file or create a new one with Ctrl+N.
  • Switch between Raw, Rendered, and Split view modes using the toolbar or Ctrl+1/2/3.

Key Features

  • Live Preview — See rendered Markdown as you type in Split or Rendered mode.
  • Syntax Highlighting — Full syntax highlighting for code blocks in 50+ languages.
  • Mermaid Diagrams — Native rendering of flowcharts and other Mermaid diagram types.
  • Integrated Terminal — Built-in terminal panel (Ctrl+`).
  • CSV/TSV Viewer — Table view for data files with sortable columns.
  • Session Restore — Automatically restores open tabs and cursor positions.
  • Multiple Themes — Light, Dark, and System theme modes.
  • Multilingual — UI available in English, Japanese, German, and more.
