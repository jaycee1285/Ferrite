<task id="2" type="code">
  <name>Implement lazy worker initialization pattern</name>
  <description>
    In `src/app.rs`, add worker handle field and lazy initialization.

    1. Add to App struct (with feature gate):
    ```rust
    #[cfg(feature = "async-workers")]
    use crate::workers::{WorkerHandle, WorkerCommand, WorkerResponse, echo_worker};

    pub struct App {
        // ... existing fields ...

        #[cfg(feature = "async-workers")]
        echo_worker: Option<WorkerHandle>,
    }
    ```

    2. Initialize in App::new():
    ```rust
    #[cfg(feature = "async-workers")]
    echo_worker: None,  // Lazy - spawns when AI panel first shown
    ```

    3. Add lazy initialization method:
    ```rust
    #[cfg(feature = "async-workers")]
    fn ensure_echo_worker(&mut self) {
        if self.echo_worker.is_none() && self.settings.ai_panel_visible {
            log::info!("Spawning echo worker (lazy initialization)");
            self.echo_worker = Some(WorkerHandle::spawn(echo_worker));

            // Process ready signal
            if let Some(worker) = &self.echo_worker {
                match worker.response_rx.try_recv() {
                    Ok(WorkerResponse::Ready) => log::info!("Echo worker ready"),
                    _ => {}
                }
            }
        }
    }
    ```

    4. Call in update() before panel rendering:
    ```rust
    #[cfg(feature = "async-workers")]
    self.ensure_echo_worker();
    ```

    This demonstrates:
    - Worker spawns ONLY when panel is first shown
    - Not on app startup (saves resources)
    - Once spawned, persists for app lifetime
  </description>

<task id="3" type="code">
  <name>Add echo demo panel (AI panel placeholder)</name>
  <description>
    In `src/app.rs`, add a temporary demo panel to prove async works.

    In App::update(), after View menu and before other panels:

    ```rust
    // Echo Demo Panel (placeholder for AI Assistant)
    #[cfg(feature = "async-workers")]
    if self.settings.ai_panel_visible {
        egui::Window::new("Echo Demo (AI Panel Placeholder)")
            .open(&mut self.settings.ai_panel_visible)
            .show(ctx, |ui| {
                ui.label("This demonstrates async workers. Type a message:");

                let mut input = String::new();
                if ui.text_edit_singleline(&mut input).lost_focus()
                    && ui.input(|i| i.key_pressed(egui::Key::Enter))
                {
                    if let Some(worker) = &self.echo_worker {
                        let _ = worker.command_tx.send(WorkerCommand::Echo(input));
                    }
                }

                ui.separator();

                // Poll for responses (non-blocking)
                if let Some(worker) = &self.echo_worker {
                    while let Ok(response) = worker.response_rx.try_recv() {
                        if let WorkerResponse::Echo(msg) = response {
                            ui.label(msg);
                        }
                    }
                }

                ui.separator();
                ui.label("This panel will be replaced with AI chat in Phase 8.");
            });
    }
    ```

    This proves:
    - Panel opens/closes via View menu
    - Worker spawns lazily
    - mpsc communication works
    - No UI blocking during async operations
  </description>
